# Notion-Style Text Editor

A production-ready, block-based rich text editor powered by Tiptap, inspired by Notion's editor UX with Microsoft Word-level formatting capabilities.

## Features

- Block-based editing with drag-and-drop reordering
- Slash command menu (/) for quick block insertion
- Bubble menu for text selection formatting
- Word-like top toolbar
- Table of Contents sidebar
- Comments/Annotations system
- Markdown and keyboard shortcuts
- Image uploads with captions
- Tables with resize support
- Mentions (@) and emoji support
- Light/dark theme support
- Auto-save functionality
- Collaboration-ready architecture

## Tech Stack

- React 18 + TypeScript
- Tiptap 2 (ProseMirror-based editor)
- Vite (build tool)
- Express (backend API)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Start backend server
npm run server

# Build for production
npm run build
```
